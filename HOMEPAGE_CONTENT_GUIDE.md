## Setup Instructions

1. **Start your Payload development server:**
   ```bash
   pnpm dev
   ```

2. **Access the admin panel:**
   Navigate to `http://localhost:3000/admin`

3. **Create or edit the Home page:**
   - Go to Collections → Pages
   - Create a new page or edit existing "Home" page
   - Set slug to `home`

4. **Add blocks in this order:**
   1. B2B Hero (or use standard Hero)
   2. How It Works
   3. Pain Points & Solutions
   4. Coverage Highlight
   5. Clinical Governance
   6. Security & Compliance
   7. Integrations List
   8. Pricing Framing
   9. Proof Strip

5. **For each block:**
   - Copy the Dutch (NL) content first
   - Save the page
   - Switch to English (EN) locale using the locale selector
   - Enter the English translations
   - Save again

6. **Publish the page** when ready

## Notes

- All blocks are designed to be responsive and accessible
- Content follows WCAG AA standards
- Trust badges and compliance information are prominently displayed
- CTAs are designed with minimum 44x44px touch targets
- All content is SEO-optimized for B2B healthcare search terms
